# kanban Client Side

Build with :
- parcel bundler
- vue
- Bootstrap-vue
- google-oauth2
- Bootstrap
- Firebase

  Demo Here: [Kanban-App](https://kanban-aarsandi.firebaseapp.com)


To run this app localy, follow this step:

-> npm install -g parcel-bundler

-> npm install

-> parcel public/index.html
